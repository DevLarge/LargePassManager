Large Pass Refactor Branch Instructions

Goal:
Practice moving from static-heavy code to instance-based services safely, without breaking your stable main branch.

1) Create and switch to a new branch
- git checkout -b refactor/instance-services

2) Push the branch to both remotes
- git push -u origin refactor/instance-services
- git push -u secondary refactor/instance-services

3) Refactor in small, safe steps
- Keep each step compiling before moving on.
- Prefer one concern per commit.

Suggested step order:
- Step A: Introduce AppPaths/AppConfig (instance object for file locations).
- Step B: Convert FileHandler static state into FileService instance.
- Step C: Keep EncryptionHandler static for now, or wrap it in CryptoService instance.
- Step D: Inject services into HubController.
- Step E: Inject services into VaultController.
- Step F: Load controllers through FXMLLoader controller factory from LargePassApp.

4) Commit frequently
- git add .
- git commit -m "Refactor FileHandler to FileService instance"

5) Keep branch updated with main
- git checkout main
- git pull
- git checkout refactor/instance-services
- git merge main

6) Verify before merge
- mvn test
- mvn javafx:run

7) Merge to main after successful tests
- git checkout main
- git merge --no-ff refactor/instance-services

8) Push main to both remotes
- git push origin main
- git push secondary main

9) Optional cleanup after merge
- git branch -d refactor/instance-services
- git push origin --delete refactor/instance-services
- git push secondary --delete refactor/instance-services

Refactor quality checklist:
- No global mutable static file fields.
- Controllers are thin: UI handling only.
- File and encryption logic live in services.
- Services are injected, not accessed globally.
- App still starts and navigates Hub -> Vault.
- First-time setup still works.
- Wrong password path still works (including data deletion behavior if kept).

If one step fails:
- Stop at that step.
- Commit any stable progress.
- Fix only the failing step before continuing.
